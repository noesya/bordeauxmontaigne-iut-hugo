Avant d'accéder aux services numériques, il est nécessaire d'activer vos deux comptes informatiques : un compte IUT et un compte Université Bordeaux Montaigne.

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 46 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: alt_background text: >-

Votre identifiant et votre clé d'activation se trouvent sur ton certificat de scolarité, partie "services numériques" !

notes: >- alt: >- credit: >- - kind: block template: title title: >- Compte IUT slug: >- compte-iut ranks: self: 2 data: layout: classic - kind: block template: timeline title: >- slug: >- ranks: self: false children: 3 data: layout: vertical events: - title: >- Étape 1 text: >-

Connectez-vous à un poste de l'IUT avec l'identifiant "inviteiut", sans mot de passe. La page "Mon compte IUT" s'ouvre automatiquement.

- title: >- Étape 2 text: >-

Entrez votre identifiant et votre clé d'activation. Choisissez un nouveau mot de passe (et ne l'oubliez pas !). Renseignez une adresse mail de secours.

- title: >- Étape 3 text: >-

Un message de confirmation apparaît en vert :) Quittez la session "inviteiut" et connectez-vous avec vos nouveaux identifiants.

- kind: block template: title title: >- Compte Université slug: >- compte-universite ranks: self: 2 data: layout: classic - kind: block template: timeline title: >- slug: >- ranks: self: false children: 3 data: layout: vertical events: - title: >- Étape 1 text: >-

Rendez-vous sur la page Mon compte de l'espace étudiant de l'Université Bordeaux Montaigne : https://etu.u-bordeaux-montaigne.fr/fr/outils/mon-compte.html

- title: >- Étape 2 text: >-

Entrez votre identifiant et votre clé d'activation. Saisissez votre nouveau mot de passe et une adresse mail de secours (de préférence les mêmes que pour l'IUT). Un message de confirmation apparaît en vert :)

- kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Une fois vos deux comptes activés, vous pourrez accéder à tous les services numériques : mail, emploi du temps, wifi, impressions...
